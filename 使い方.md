# BrainCare 使い方ガイド

このドキュメントは、BrainCareを実際に施設で使うための手順をまとめたものです。開発者向けの仕様は「仕様書」を参照してください。

---

## 1. 初回セットアップ（導入担当者向け）

### 必要なもの

* Ubuntu Linux（またはPHP 8.1以上が動く環境）
* MySQL または MariaDB
* Composer（`composer.phar`はこのプロジェクトに同梱済み）

### 手順

1. **依存パッケージを導入する**

   ```
   php composer.phar install
   ```

2. **データベースを用意する**

   ```
   sudo mysql -e "CREATE USER IF NOT EXISTS 'braincare'@'127.0.0.1' IDENTIFIED BY 'braincare'; GRANT ALL PRIVILEGES ON braincare.* TO 'braincare'@'127.0.0.1'; FLUSH PRIVILEGES;"
   sudo mysql < database/braincare.sql
   ```

   `database/braincare.sql`には、初期管理者アカウント（名前:`admin` / パスワード:`admin1234`）が自動で登録されます。**初回ログイン後、必ずパスワードを変更してください。**

3. **接続設定を確認する（必要に応じて変更）**

   `config/config.php`にDB接続情報とWebSocketのポート設定があります。環境変数（`BRAINCARE_DB_HOST`など）で上書きできます。本番運用では環境変数での設定を推奨します。

4. **サーバーを起動する**

   開発・検証用の簡易起動：

   ```
   php -S 0.0.0.0:8000 &
   php ws/server.php &
   ```

   本番運用時は、Nginx + PHP-FPMでHTTP（`php/`配下）を配信し、`ws/server.php`はsystemdサービスとして常駐させることを推奨します（`仕様書`の該当箇所を参照）。

5. **各画面のURLをブラウザで開く**

   * テレビ側：`http://<サーバーのIP>:8000/main.html`
   * スマホ側：`http://<サーバーのIP>:8000/konto.html`
   * 管理画面：`http://<サーバーのIP>:8000/admin.html`

   konto.html／main.htmlの先頭にある`<meta name="braincare-ws-url" ...>`のURLが、実際のWebSocketサーバーのアドレスと合っているか確認してください（既定は`ws://localhost:8080`）。

## 2. 管理者（介護スタッフ）の使い方

### ログイン

`admin.html`を開き、名前とパスワードでログインします（利用者と違い、管理者は必ずパスワードが必要です）。

### 利用者を登録する

1. 左メニューの「利用者登録」を開く
2. 権限は「利用者」を選ぶ（パスワード欄は自動的に隠れます。利用者はパスワード不要のため）
3. お名前・生年月日（任意）を入力して「登録する」

これで、その利用者はスマートフォンのログイン画面に名前が表示されるようになります。

### 利用状況を確認する

* **概要・統計**：利用者数、プレイ回数、ゲーム種別ごとの利用状況グラフ
* **利用者一覧**：登録済みの利用者と、各利用者の「個人の記録を見る」ボタン
* **個人の記録**：特定の利用者について、プレイ回数・平均スコア・連続プレイ日数・獲得バッジ・今日の目標達成状況・苦手分野グラフ・直近の履歴を確認できる
* **学習履歴／対戦履歴／ランキング**：全利用者分の一覧

### 新しい管理者を追加する場合

「利用者登録」で権限を「管理者」に切り替えるとパスワード欄が表示されます。パスワードは4文字以上で設定してください。

## 3. 利用者（ご本人）の使い方

### ログイン

`konto.html`（スマートフォン）を開き、一覧から自分の名前をタップするだけです。パスワードの入力は不要です。

複数のテレビがある施設では、画面下の「画面番号の設定」を開いて、遊びたいテレビの番号を選んでください（1台だけの場合は設定不要です）。

### 一人で遊ぶ

メニューの「一人で遊ぶ」から、好きなゲームを選びます。

| ゲーム名 | 内容 |
| --- | --- |
| 計算問題 | 四則演算 |
| 記憶ゲーム | 数字・色の並びを覚えて答える |
| 数字順タッチ | 数字を小さい順にタップする |
| 文字並べ替え | バラバラの文字を並べて言葉を完成させる |
| ○×クイズ | 文章が正しいか答える |
| 間違い探し | 2枚の絵を見比べて違いをタップする |

テレビに問題が表示されるので、スマートフォンで回答します。5問終わると結果が表示されます。

### 認知機能テストを受ける

メニューの「認知機能テストを受ける」を選ぶと、上記6種類のゲームが1問ずつ順番に出題されます。終わると「苦手分野を見る」ボタンから、どの分野が苦手かをすぐに確認できます。

### 苦手克服モード

自分の苦手なゲームを自動的に選んでソロプレイを始めるボタンです。何を選べばよいか迷ったときに便利です。

### みんなで遊ぶ（対戦）

1. テレビ側で、介護スタッフまたは利用者がゲームを選ぶと、テレビに4桁の「参加コード」が表示されます。
2. 対戦したい2人が、それぞれのスマートフォンで「みんなで遊ぶ」を選び、そのコードを入力します。
3. 2人目が入力すると自動的に対戦が始まります。同じ問題に同時に回答し、正解数とスコアを競います。

### 自分の記録を見る

* プレイ回数、平均スコア、最高スコア、連続プレイ日数
* 対戦成績（勝敗数）とランキング順位
* 獲得したバッジ（🥇初プレイ／🥈100問正解／🏆7日連続プレイ／🎉対戦10勝）
* 苦手分野グラフ（正答率が低い順に表示。赤色ほど苦手、緑色ほど得意）
* 直近のプレイ履歴

### 今日の目標

メニュー画面の上部に、その日の目標（3ゲーム遊ぶ／計算問題を1回行う／認知機能テストを受ける）と達成率が表示されます。毎日プレイする習慣づけに使ってください。

### 音声案内

メニュー画面の「🔊音声案内」ボタンでON/OFFを切り替えられます（初期状態はON）。ONにしていると、問題文や正解・不正解の結果を音声で読み上げます。設定はそのスマートフォン（ブラウザ）に保存されます。

---

## 4. テレビ（大型モニター）の使い方

`main.html`を開いておくだけで、回答入力は不要です。

* 待機中は、施設全体のランキングと「みんなで遊ぶゲームを選ぶ」ボタンが表示されます
* 誰かがソロプレイ・対戦・認知機能テストを始めると、自動的に問題画面に切り替わります
* 終了後、8秒ほどで待機画面に戻ります

## 5. よくある質問

**Q. スマホのログイン画面に自分の名前が出てこない**
A. 介護スタッフに利用者登録を依頼してください（管理画面の「利用者登録」）。

**Q. 参加コードを入力しても「見つかりません」と出る**
A. コードには有効期限があるわけではありませんが、対戦が始まると同じコードは無効になります。テレビ側でもう一度ゲームを選び直し、新しいコードを確認してください。

**Q. 音声が出ない**
A. メニュー画面の「🔊音声案内」がOFFになっていないか確認してください。また、スマートフォンの音量設定や、ブラウザが音声合成に対応しているかもご確認ください。

**Q. 管理者パスワードを忘れた**
A. データベースを直接操作してパスワードを再設定する必要があります（開発担当者にご連絡ください）。

**Q. 対戦の途中で相手が切断してしまった**
A. 残った側の勝利として自動的に記録され、対戦は終了します。
